# CodeCJK

CodeCJK is a font file combining Consola and NotoSansMonoCJKhk-Regular, generated by mono-merge.

## How can I download it?

https://www.dropbox.com/scl/fo/b7ge8rq1qrsyc2ogckm9m/AKgcHOA4FOL8SWtWJpJM1SI?rlkey=5r2xlds9ar2g3f18obl3b7m5t&st=tzueh14z&dl=0

The file is not put to GitHub directly because there is license issue.

## How to create the ttf file?

```bash
# From Windows 11, version 7.00
md5sum input/consola.ttf
# 5715d3f0bfcd28a4e008721a66118fe2  input/consola.ttf

# From https://github.com/notofonts/noto-cjk/tree/f8d157532fbfaeda587e826d4cd5b21a49186f7c/Sans/Mono
md5sum input/NotoSansMonoCJKhk-Regular.otf 
# ad5f30522e2281bfb7f3d37a8605253c  input/NotoSansMonoCJKhk-Regular.otf

python3 monomerge.py -l input/consola.ttf -c input/NotoSansMonoCJKhk-Regular.otf --cjk-y-offset=-81 -o output/CodeCJK002_DATETIME.ttf -n CodeCJK002

python3 monomerge.py -l input/consola.ttf -c input/NotoSansMonoCJKhk-Regular.otf --cjk-y-offset=-81 -o output/CodeCJK_DATETIME.ttf -n CodeCJK
```
